# Impostazioni Articoli

## Aggiorna automaticamente costo da carico

Se attivo questo flag, quando si registra un carico (per esempio una fattura fornitore ) il costo indicato per l'acquisto di un articolo verrà riportato nella scheda articolo, nel campo Costo, a condizione che la data del documento di carico sia uguale o successiva alla data di ultimo carico (Riportata nella tab Magazzino dell'anagrafica articolo)
